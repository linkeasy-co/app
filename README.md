# Meu App React

Bem-vindo ao **Meu App React**! Este é um projeto criado com [Create React App](https://create-react-app.dev/). Certifique-se de ter as seguintes ferramentas instaladas antes de começar: **Node.js** versão **23** ou superior e [Create React App (CRA)](https://create-react-app.dev/). Siga os passos abaixo para rodar o app no seu ambiente local: clone este repositório usando `git clone <URL_DO_REPOSITORIO>`, navegue até o diretório do projeto com `cd meu-app-react`, instale as dependências do projeto rodando `npm install` e, em seguida, inicie o servidor de desenvolvimento com `npm run start`. O app estará disponível no navegador em **http://localhost:3000**.

No diretório do projeto, você pode rodar os seguintes comandos: `npm start` para iniciar o servidor de desenvolvimento, `npm test` para rodar os testes no modo interativo e `npm run build` para gerar a versão otimizada do app para produção.

A estrutura básica do projeto é organizada assim: `meu-app-react/` contém as pastas `public/`, `src/` (com subpastas como `components/`, `pages/`, além dos arquivos `App.js` e `index.js`) e o arquivo `package.json`.

Tecnologias usadas: **React**, **Node.js** e **Create React App**.
